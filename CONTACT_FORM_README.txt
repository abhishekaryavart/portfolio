# 🚀 Contact Form Setup - Quick Start

Your portfolio's contact form is now powered by **Formspree** for GitHub Pages compatibility!

## ⚡ Quick Setup (5 minutes)

### Step 1️⃣: Get Your Form ID
1. Go to **https://formspree.io**
2. Sign up (free)
3. Create a new form
4. Copy the **Form ID** (e.g., `mpppknep`)

### Step 2️⃣: Update Your Portfolio
1. Open `script.js`
2. Find line ~225: `const formId = 'xyzabc';`
3. Replace `'xyzabc'` with your Form ID
4. Save & deploy!

### Step 3️⃣: Test It! ✅
Send a test message from your portfolio - it should arrive in your inbox!

## 🔧 Need Help?

**Form ID not working?**
- Verify your Formspree account email is confirmed
- Double-check the Form ID is copied exactly
- Try a fresh form on Formspree

**Not receiving emails?**
- Check Formspree dashboard "Submissions" tab
- Verify destination email in account settings
- Check spam/junk folder

**Questions?** Visit https://formspree.io/help
