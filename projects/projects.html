DSVV Quiz Master

The DSVV Quiz Master is a comprehensive real-time quiz application designed for hosting large-scale quiz competitions. Built on a Flask/Python backend, it leverages MongoDB for persistent data storage and Flask-SocketIO for real-time bidirectional communication between the host and participants.

Technologies Used

Python, Flask, Socket.io, MongoDB, HTML5, CSS3, JavaScript
